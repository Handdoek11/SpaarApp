## Usage Examples

### Development Workflow
```bash
# Start development with AI assistance
claude-code spaarapp:dev

# Run tests during development
claude-code spaarapp:test

# Check security before commits
claude-code spaarapp:security
```

### Release Workflow
```bash
# Full analysis before release
claude-code spaarapp:analyze

# Optimize performance
claude-code spaarapp:optimize

# Security audit
claude-code spaarapp:security

# Build and package
claude-code spaarapp:deploy
```

### Maintenance Workflow
```bash
# Regular security checks
claude-code spaarapp:security

# Performance monitoring
claude-code spaarapp:analyze

# Accessibility validation
claude-code spaarapp:test
```

These commands provide AI-enhanced development workflows specifically designed for building an ADHD-friendly financial application with Dutch market compliance and enterprise-grade security.
